Project Title :- Road Vehicle Breakdown Assistance System – Backend
1) Project Description:-

The Road Vehicle Breakdown Assistance System is a backend application designed to manage vehicle breakdown service requests and connect users with available mechanics.

The system follows a microservices architecture where independent services handle different responsibilities such as user management, mechanic operations, and administrative monitoring.

This backend system is developed using Spring Boot, REST APIs, Microservices Architecture, and MySQL.

2)System Architecture

The backend consists of three independent microservices:

-User Service

-Mechanic Service

-Admin Service

Each service:

Runs independently

Exposes REST APIs

Maintains its own MySQL database
